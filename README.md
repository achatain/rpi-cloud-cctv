Raspberry Pi Cloud CCTV
=========================

https://github.com/achatain/rpi-cloud-cctv

## What is it?

This repo hosts a Python app powering a DIY CCTV made of a Raspberry Pi, a camera module and a PIR (Passive Infrared) motion sensor.
